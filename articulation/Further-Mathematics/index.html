# 极限的运算方法

  • lim⁡x→∞axm+bxncxp+dxq\lim\limits_{x \to \infin} \frac{ax^m + bx^n}{cx^p + dx^q}x→∞lim​cxp+dxqaxm+bxn​

  • 比较 m、n、p、q 中的最大值，然后分子分母同时除以最高项

  • 如果分子分母的最高次项相同，那么极限即为最高次项前面常数之比；如果分子最高次项比分母最高次项小，那么极限为 0

  • 例题 1：lim⁡x→∞3x2+5x2x2+4x+5\lim\limits_{x \to \infin} \frac{3x^2 + 5x}{2x^2 + 4x + 5}x→∞lim​2x2+4x+53x2+5x​，答案为 32\frac{3}{2}23​

  • 例题 2：lim⁡x→∞4x2−2x2+92x3−5x2+7\lim\limits_{x \to \infin} \frac{4x^2 - 2x^2 + 9}{2x^3 - 5x^2 + 7}x→∞lim​2x3−5x2+74x2−2x2+9​，答案为 0
